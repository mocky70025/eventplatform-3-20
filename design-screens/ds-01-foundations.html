1. Color

出店者=Green、主催者=Orangeをプライマリとし、Neutralとアクセントカラーを共有する。同一プラットフォームの一体感を保ちながら、ロールの区別を色で直感的に伝える。

意図: 安心感、自然、信頼。出店者が「安心してビジネスを任せられる」印象を与える。

2. Typography

Noto Sans JP一本。ウェイトとサイズの組み合わせで階層を作る。日本語の可読性を最優先し、行間は1.6--1.8を基本とする。

Token Size / Weight Line Height Sample 用途
display 48px / 900 1.2 楽しさを、ここから Hero見出し（1画面に1箇所のみ）
heading-1 30px / 700 1.3 注目のイベント ページタイトル
heading-2 24px / 700 1.4 イベント概要 セクション見出し
heading-3 18px / 600 1.5 基本情報 カード見出し、サブセクション
body 15px / 400 1.7 全国のイベント情報を検索して、ワンクリックで出店申し込み。 本文、説明テキスト
body-sm 14px / 500 1.6 申込日: 2026年3月10日 ラベル、メタ情報、ナビゲーション
caption 12px / 400 1.5 残り12日 / JPG, PNG (最大10MB) 補足、ヒント、バッジ内テキスト

3. Spacing & Radius

4pxベースのスケール。余白が広いほどリラックスした印象になり、イベントの楽しさに合う。角丸は大きめに設定し、硬さを取り除く。

4. Layout

全画面1440px幅を基準とし、コンテンツ幅はmax-widthで制御。ページ種別ごとに適切な幅を使い分ける。

max-w-7xl (1280px) ホーム、イベント一覧

5. Gradients

「楽しさ」を視覚的に伝える装飾グラデーション。使用箇所を限定し、乱用しない。

Fun Gradient (Store)

用途: Hero背景、Auth左パネル
Green → Yellow → Pink

Fun Gradient (Organizer)

用途: Hero背景、Auth左パネル
Orange → Yellow → Pink

Store Dark

用途: 統計バー、フッター
Green-500 → Green-700

Organizer Dark

用途: 統計バー、フッター
Orange-500 → Orange-700

Image Placeholder Gradients

イベント画像が未設定の場合に使用。カテゴリごとに異なるグラデーションで視覚的な区別を提供。
